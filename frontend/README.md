# Claude Optimize Frontend

This package contains the React + TypeScript + Vite frontend for Claude Optimize.

## Scripts

- `npm run dev`: start the local Vite dev server
- `npm run build`: run TypeScript compilation and create a production bundle
- `npm run lint`: run ESLint on the frontend source
- `npm run preview`: preview the production bundle locally

## Notes

- During local development, Vite proxies `/api` requests to `http://localhost:8000`.
- The full-stack setup and backend instructions live in the repository root `README.md`.
